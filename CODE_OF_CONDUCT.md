# Code of Conduct

This project follows a simple standard: be respectful, be constructive, and
assume good faith.

- Disagreements about code and design are welcome; personal attacks are not.
- Harassment, discriminatory language, or trolling of any kind will not be
  tolerated in issues, pull requests, or any other project space.
- Maintainers may remove comments, commits, or contributors that violate this
  standard, and may report unacceptable behavior as appropriate.

To report a concern, open an issue or contact the maintainer directly through
their GitHub profile.

This is a condensed policy in the spirit of the
[Contributor Covenant](https://www.contributor-covenant.org/).
