# Security Policy

## Supported Versions

Only the latest published version on npm receives security fixes.

## Reporting a Vulnerability

This package has zero runtime dependencies, which keeps the attack surface small,
but if you find a security issue:

1. Use GitHub's [private vulnerability reporting](https://github.com/donizetiferr/forecast-calibration-kit/security/advisories/new)
   for anything sensitive.
2. Otherwise, open a regular [issue](https://github.com/donizetiferr/forecast-calibration-kit/issues).

Please include a minimal reproduction and the affected version. Expect an initial
response within a few days.
